Autofill Service

Increase your productivity by allowing Password Vault to automatically fill login data into other apps. This page contains information on how to configure this service.

Enable Autofill Service

In order to enable the autofill service, go to Settings > Autofill. If the autofill service is not yet enabled, the page will display a button Enable autofill:

Click this button. This opens your Android device's settings. Within the settings, select Password Vault.

Afterwards, Password Vault is selected as autofill service.

Prepare Accounts for Autofill

In order to autofill, Password Vault needs to know which account to use for which of your installed apps. This can be configured when you edit the respective account.

Click on the edit-button to select an installed app:

Next, switch to Installed apps and select the apps which you want to autofill. You can select multiple apps if you want.

Afterwards, close the page. And click the save-button.

Which Details are Selected?

Usually, you need some sort of username and a password to login to an app. From all provided details, Password Vault automatically selects a suitable username and password to autofill. However, if you want to use a specific detail as username or password, you can select the respective option when editing the detail:

A username selected in this manner is indicated through a small e-mail icon. A password selected this way is indicated through a small password icon.

If you select multiple details as username or password, the first one will be used for autofill.

Authentication

Cache

Caching accelerates the autofill service. This is done by storing your login data in a separate (encrypted) file. Therefore, Password Vault does not need to search through the entire app data every time you use autofill. However, you can disable this option.
