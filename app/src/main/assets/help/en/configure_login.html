App Login

Increase the security of the sensitive data stored in Password Vault by enabeling the app login through a master password.

Table of Contents

  1. Enable App Login
  2. Enable Biometrics

Enable App Login

In order to enable the app login, go to Settings > Security and click on the Require login-switch:

This opens a dialog in which you need to enter a master password. Enter and confirm the password. Click the Ok-button afterwards:

With this, the login is enabled. You will need this master password whenever you open the app. Do not forget the password under any circumstances, since the password cannot be reset!

Enable Biometrics

If your device supports class 3 biometrics, a Use biometrics-switch appears. Click this switch to enable biometrics:
