Backups

Erstelle Backups um Daten aus Password Vault zwischen Geräten zu transferieren oder um deine Daten auf ein anderes Gerät zu verschieben, falls du dein Android-Smartphone verlierst.

Inhaltsverzeichnis

  1. Backup erstellen
  2. Backup verschlüsseln
  3. Backup wiederherstellen

Backup erstellen

Um ein Backup zu erstellen, gehe zu Einstellungen > Daten und klicke auf den Backup erstellen-Knopf:

Danach öffnet sich ein neues Fenster, über welches das Backup konfiguriert werden kann. Klicke auf das Eingabefeld "Ordner" in der oberen Hälfte des Bildschirmes um einen Ordner auszuwählen, in welchem das Backup gespeichert werden soll:

Es öffnet sich der Dateiexplorer deines Smartphones. Navigiere zum Ordner, in welchem du das Backup speichern möchtest und klicke auf den Diesen Ordner verwenden-Knopf. Unter Umständen musst du Password Vault die Erlaubnis erteilen, um Dateien in dem ausgewählten Ordner zu erstellen. Klicke dazu auf Zulassen:

Falls du die App-Einstellungen oder die Passwortsicherheitskritereien in dem Backup exportieren möchtest, dann klicke auf die entsprechenden Schalter:

Klicke danachauf den Backup erstellen-Knopf:

Backup verschlüsseln

Um ein Backup zu verschlüsseln, klicke auf den entsprechenden Schalter und gebe ein Password an:

Das Passwort wird verwendet, um das Backup zu verschlüsseln. Wenn das Backup wiederhergestellt werden soll, muss dieses Passwort angegeben werden. Andernfalls kann das Backup nicht entschlüsselt werden.

Backup wiederherstellen

Um ein Backup wiederherzustellen, gehe zu Einstellungen > Daten und klicke auf Backup wiederherstellen:

Danach öffnet sich der Dateiexplorer deines Android-Gerätes. Navigiere zum Ordner, in welchem das wiederherzustellende Backup gespeichert ist und klicke auf die entsprechende Datei:

Es öffnet sich ein Fenster, über welches das Backup wiederhergestellt werden kann. Falls möglich, wähle aus, was mit den bestehenden Daten in Password Vault durchgeführt werden soll.

Folgende Möglichkeiten stehen zur Verfügung:

  • Alle vorhandenden Daten löschen: Hierdurch werden alle Daten, die aktuell in Password Vault gespeichert sind gelöscht. Danach wird das Backup wiederhergestellt.

  • Duplikate überschreiben: Falls einige Daten bereits in Password Vault gespeichert sind, werden diese durch die Daten im Backup überschrieben.

  • Duplikate überspringen: Falls eine Daten bereits in Password Vault gespeichert sind, werden die jeweiligen Daten aus dem Backup nicht wiederhergestellt.
